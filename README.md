## ❓ COMMON QUESTIONS

**Q: Can I modify the dashboard?**  
A: Yes. Source code is in `/home/claude/amgen-fin/`:
- `build_data.py` — Generates data.json
- `build_html.py` — Generates index.html
- Rebuild with: `python3 build_data.py && python3 build_html.py`

**Q: Is this real data?**  
A: Yes. 100% sourced from Amgen's FY2024 10-K (filed Feb 14, 2025, SEC EDGAR). No synthetic/estimated data (except forward-looking scenarios).

**Q: Can I share this externally?**  
A: Yes. Dashboard is self-contained HTML (no API dependencies). Attach with email or post to internal portal. Recommend including the Integration Guide.

**Q: How do I print this for Board meeting?**  
A: Use browser Print function (Ctrl+P / Cmd+P). Landscape orientation recommended. Charts render cleanly at 100% zoom. Alternatively, screenshot individual sections.

**Q: Who owns the implementation?**  
A: **Finance leadership** sets target; **Supply Chain + Treasury** execute payables/inventory; **Commercial** executes collections (DSO); **Operations** executes manufacturing/resilience.

**Q: What if execution is slower than 24 months?**  
A: Still valuable. Even 50% mitigation execution = $600M-$1B WC release. Phased approach is fine; just extend timeline proportionally.
